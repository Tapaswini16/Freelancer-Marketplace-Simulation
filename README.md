# Freelancer Marketplace Simulation (Python)

This is a simple Python project that simulates a basic freelancer marketplace system.
The program allows users to register freelancers and clients, assign projects, and process payments.

## Features

* Register Freelancer with skills
* Register Client
* Create and assign projects
* Process payment to freelancer

## Technologies Used

* Python
* Object-Oriented Programming (OOP)

## Classes Used

* Freelancer
* Client
* Project

## How to Run

1. Download or clone the repository.
2. Run the Python file.

```
python freelancer_marketplace.py
```

3. Enter the required details when prompted.

## Example

The program will ask for freelancer name, skill, client name, project title, and budget, then assign the project and process payment.
